# 北京高一物理上册 · 精华交互学习站

纯前端（HTML + CSS + JS），无需安装，双击打开即可用。也可通过妙搭发布外网访问。

## 打开方式

1. 进入文件夹 `beijing-physics-g10-1`
2. 双击 **`index.html`**，用浏览器打开（推荐 Chrome / Edge）

或在终端：

```powershell
start C:\Users\itoulee\beijing-physics-g10-1\index.html
```

## 内容完整度

**精华 + 扩充学习站**，不是教材逐页原文。覆盖北京高一上册（人教版必修第一册）主干考点与方法；不含全套课后题详解。

## 内容结构

### 主线精讲
| 章节 | 内容 |
|------|------|
| 运动的描述 | 质点、参考系、位移、速度、加速度 |
| 匀变速直线运动 | v-t / x-t、三大公式、自由落体 |
| 相互作用 | 重力、弹力、摩擦、力的合成与分解 |
| 牛顿运动定律 | 三大定律、单位制、超重失重 |

### 扩充学习
| 章节 | 内容 |
|------|------|
| 典型例题精讲 | 5 道高频题分步展开 |
| 易错点与方法卡 | 陷阱清单 + 闪卡自测 |
| 综合自测 | 12 题跨章检测 |

## 交互实验

- v-t 图线实验室（读面积 = 位移）
- 自由落体仿真
- 力的合成（平行四边形）
- 斜面摩擦力
- 牛顿第二定律 F=ma
